COMMAND USAGE: MPI.Init()

MPI.Init() is called when MPI is imported from the mpi4py package. No need to initialize in python.

MPI COMM SIZE and RANK

  • MPI_COMM_SIZE returns the number of processors available.
  • MPI_COMM_RANK returns the current processor number.

  • Fortran 95
  • Python

COMMAND USAGE: call MPI_COMM_SIZE( comm, nproc, ierr )

COMMAND USAGE: call MPI_COMM_RANK( comm, iproc, ierr )

  • comm: Communicator that is holding all the processes.
  • nproc: INTEGER that will hold the number of processors.
  • iproc: INTEGER that will hold the current processor identifier.
  • ierr: Holds the error code, if any.

COMMAND USAGE: comm.size

COMMAND USAGE: comm.Get_rank()

  • comm.size: returns the number of processors being used.
  • comm.Get_rank(): returns the current processor identifier.

Broadcasting data

Data is sent to all other processors, while the processor sending the data keeps a copy of the data.

  • Fortran 95
  • Python

COMMAND USAGE: call MPI_BCAST( buffer, buffer size, MPI_TYPE, root, comm, ierr )

  • buffer: data that is being sent. Can be a single number or an array of any size.
  • comm: Communicator that is holding all the processes.
  • nproc: INTEGER that will hold the number of processors.
  • iproc: INTEGER that will hold the current processor identifier.
  • ierr: Holds the error code, if any.

COMMAND USAGE: buffer_out = comm.bcast( buffer_in, root=0 )

  • comm = MPI.COMM_WORLD
  • buffer_in/out: data that is being sent/received. Can be a single number or an array of any size.
  • root: The number of the main processor.

Reducing data

Application of operation on a variable, if single variable, or on matching indicies, if array.

  • Fortran 95
  • Python

COMMAND USAGE: CALL MPI_REDUCE( buffer, second buffer, size, MPI_TYPE, MPI_CALC, root, comm, ierr )

  • buffer: variable/array that will be operated on.
  • second buffer: buffer that will hold the final value.
  • size: size of buffer. second buffer must be the same size.
  • MPI_TYPE: data type that the buffers hold.
  • MPI_CALC: calculation that will be applied to each element.
  • root: root processor integer value.
  • comm: Communicator that is holding all the processes.
  • ierr: Holds the error code, if any.

COMMAND USAGE: second buffer = comm.reduce( buffer, op=operation, root=0 )

  • second buffer: buffer that will hold the final value.
  • buffer: variable/array that will be operated on.
  • operation: operation that will be applied to each element.
  • root: root processor integer value, usually 0.
