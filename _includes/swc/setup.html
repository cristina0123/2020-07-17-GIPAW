pw.x input

  • &control card
    • calculation: A string describing the task that will be performed but for the purposes of this tutorial we will only be using 'scf'
    • prefix: Prepended to input/output file names and extremely important that file name is unique and the same in the gipaw input file
    • pseudo_dir: Directory that contains the pseudopotential files
    • outdir: Temporary output directory
  • &system card
    • ibrav: The bravais-lattice structure of the system
    • celldm(#): Based on the bravais-lattice structure
      • 1 = a
      • 2 = b/a
      • 3 = c/a
      • 4 = cos(bc)
      • 5 = cos(ac)
      • 6 = cos(ab)
    • nat: Number of atoms in the unit cell
    • ntyp: Number of types of atoms in the unit cell
    • ecutwfc: Kinetic eneryg cutoff (Ry) for the wavefunctions
      • It is normal to start off with 90.0 but then to find the convergence for the energy
  • ATOMIC_SPECIES card

    • This is a 3 column list of the atom type, mass of the atomic species, and the file containing the pseudopotential for this species:

    X Mass_X PseudoPot_X

  • ATOMIC_POSITIONS {XXXX} card: the card has several description options and it is based on the units for the atomic postions

    • This is a 4 column list of the atom type and then its x, y, z atomic positons:

    X x y z

gipaw.x input

&inputgipaw card

  • job: A string describing the task that will be performed but for the purposes of this tutorial we will only be using 'nmr'
    • Types of jobs:
      • nmr: magnetic susceptibility & sheilding tensors
      • efg: electric field gradients (EFGs)
      • g_tensor: electron paramegnetic resonance (EPR) g-tensor
      • hyperfine: hyperfine couplings
  • prefix: Prepended to input/output file names and extremely important that file name is unique and the same in the pw.x input file
  • tmp_dir: Temporary output directory
