MPI_INIT

Initializes the parallel processes.

  • Fortran 95
  • Python

COMMAND USAGE: call MPI_INIT(ierr)

ierr is an integer that contains the error code.

COMMAND USAGE: MPI.Init()

MPI.Init() is called when MPI is imported from the mpi4py package. No need to initialize in python.

MPI COMM SIZE and RANK

MPI_COMM_SIZE returns the number of processors available. MPI_COMM_RANK returns the current processor number.

  • Fortran 95
  • Python

COMMAND USAGE: call MPI_COMM_SIZE( comm, nproc, ierr )

COMMAND USAGE: call MPI_COMM_RANK( comm, iproc, ierr )

comm: Communicator that is holding all the processes. nproc: INTEGER that will hold the number of processors. iproc: INTEGER that will hold the current processor identifier. ierr: Holds the error code, if any.

COMMAND USAGE: comm.size

COMMAND USAGE: comm.Get_rank()

comm.size: returns the number of processors being used. comm.Get_rank(): returns the current processor identifier.
